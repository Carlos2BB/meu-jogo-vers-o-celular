ARCADE PARK CHALLENGE

Seja bem vindo ao meu primeiro jogo, vc tem o objetivo de procurar moedas pelo mapa para pegar os premios de uma maquina para ganhar, so que não é tão facil assim, no jogo existem nextbot que vem atras de vc para não deixar vc completar o desafio.

No jogo existem alguns poderes, são eles:
A Faca: serve para vc destrancar uma barreira que vai estar em volta da maquina de pegar premios para ai sim vc poder usar ela.
A Arma: serve para vc paralisar os nextbot por 5 segundos para ficar mais facil de fugir.
As Botas: serve para vc ter ascesso ao parkour, e no final dele tem um báu com 10 moedas.

Se vc não quiser usar a faca, tem tambem a opção de usar o R para desbloquear a máquina (leva 30 segundos). Esse é um jogo de quem termina mais rapido, por isso do cronometro. Lembrando: é o meu primeiro jogo e não vai estar 100% perfeito. Boa sorte!
